# Phase 1 Business App (GitHub Pages + Google Sheets)

## What you get in Phase 1
- Dashboard (last 30 days stats)
- Quotation: create/edit, status Draft/Pending/Cancelled, clean print
- Quotations list: filter + status update + convert to invoice
- Invoice (Sales): create invoice by converting quotation
- Dispatch: create dispatch from invoice and track Delivered / Delayed

## Setup
1) Deploy Apps Script as Web App (Execute as Me, Access Anyone)
2) Copy Web App URL (ends with /exec)
3) Paste into api.js -> API_BASE
4) Upload all files to GitHub repo root and enable GitHub Pages

Next phases:
- Phase 2: Inventory ledger + low stock + dead stock + item categories
- Phase 3: Payments + debtors (good/bad) + credit risk
